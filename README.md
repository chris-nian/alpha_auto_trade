# 币安Alpha自动交易插件

一个专为币安Alpha交易页面设计的Chrome浏览器扩展插件，能够自动执行买入-卖出交易循环。

## 功能特性

### 🚀 核心功能
- **自动交易循环**：自动执行买入→等待成交→卖出→等待成交的完整交易流程
- **智能订单监控**：实时监控订单状态，确保买入/卖出操作完成后再进行下一步
- **交易次数控制**：支持设置交易次数限制，达到限制后自动停止并安全卖出
- **金额自定义**：可设置每次交易的USDT金额（最小0.1 USDT）

### 🛡️ 安全保护
- **页面验证**：仅在币安Alpha交易页面生效
- **登录检查**：自动检测用户登录状态
- **网络监控**：实时检查网络连接状态
- **安全停止**：一键安全停止，自动切换到卖出并清仓
- **错误重试**：内置错误重试机制，提高交易成功率

### 📊 界面功能
- **实时状态显示**：显示当前交易状态和进度
- **交易计数器**：实时显示已完成的交易次数
- **详细日志**：保存最多200条交易日志，便于追踪交易过程
- **可拖拽界面**：支持拖拽移动，不影响正常浏览
- **最小化功能**：支持界面最小化，节省屏幕空间

## 安装说明

### 1. 下载插件文件
确保您有以下文件：
- `manifest.json` - 插件配置文件
- `popup.html` - 弹窗界面
- `popup.js` - 弹窗逻辑
- `content.js` - 核心交易逻辑
- `styles.css` - 样式文件
- `icon.png` - 插件图标

### 2. 安装到Chrome
1. 打开Chrome浏览器
2. 地址栏输入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择插件文件所在的文件夹
6. 插件安装完成

### 3. 权限说明
插件需要以下权限：
- `activeTab`：访问当前活跃标签页
- `storage`：保存用户设置
- `https://www.binance.com/*`：仅在币安网站运行

## 使用指南

### 基本使用流程

1. **打开交易页面**
   - 登录币安账户
   - 导航到Alpha交易页面：`https://www.binance.com/zh-CN/alpha/`
   - 选择要交易的代币

2. **配置插件**
   - 点击浏览器工具栏中的插件图标
   - 设置交易金额（USDT）
   - 设置交易次数限制（0表示无限制）

3. **开始交易**
   - 点击"开始自动交易"按钮
   - 插件会自动开始执行交易循环
   - 可通过页面内嵌的控制面板实时监控

### 控制面板功能

#### 主要按钮
- **开始交易**：启动自动交易流程
- **停止交易**：正常停止交易（完成当前订单后停止）
- **安全停止交易**：立即停止所有活动并自动卖出所有代币

#### 信息显示
- **状态显示**：当前交易状态（等待开始/交易运行中）
- **交易计数器**：显示已完成交易次数和设置的限制
- **实时日志**：显示详细的交易操作记录

#### 工具功能
- **清空日志**：清除所有日志记录
- **拖拽移动**：点击标题栏可拖拽移动界面位置
- **最小化**：点击"-"按钮最小化界面

### 交易流程详解

1. **买入阶段**
   - 自动切换到买入选项卡
   - 设置指定的交易金额
   - 点击买入按钮并确认订单
   - 监控买入订单直到完成

2. **卖出阶段**
   - 自动切换到卖出选项卡
   - 设置100%数量（全部卖出）
   - 点击卖出按钮并确认订单
   - 监控卖出订单直到完成

3. **循环控制**
   - 每完成一轮买卖记录为1次交易
   - 达到设置的交易次数限制后自动停止
   - 自动执行安全卖出以避免持仓风险

## 注意事项

### ⚠️ 重要警告
1. **风险提示**：加密货币交易存在高风险，请确保您了解相关风险
2. **资金安全**：建议使用少量资金进行测试
3. **市场波动**：插件无法预测市场走势，请谨慎使用
4. **网络稳定**：确保网络连接稳定，避免交易中断

### 📋 使用建议
1. **首次使用**：建议先用小额资金测试插件功能
2. **监控交易**：使用过程中请密切关注交易日志和订单状态
3. **及时停止**：发现异常情况请立即使用"安全停止交易"功能
4. **定期检查**：定期检查账户余额和交易记录

### 🔧 故障排除

#### 常见问题
1. **插件无法启动**
   - 确认是否在正确的币安Alpha交易页面
   - 检查是否已登录币安账户
   - 刷新页面后重试

2. **买入/卖出失败**
   - 检查账户余额是否充足
   - 确认网络连接稳定
   - 查看页面是否有错误提示

3. **订单长时间未完成**
   - 检查市场流动性
   - 考虑调整交易金额
   - 使用"安全停止交易"功能

#### 调试信息
- 插件会在浏览器控制台输出详细的调试信息
- 按F12打开开发者工具查看Console选项卡
- 日志前缀为`[Binance Auto Trader]`
